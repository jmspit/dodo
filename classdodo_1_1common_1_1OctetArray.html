Detailed Description

An array of Octets with size elements.

Provides base64 conversion, random data generation, appending of data from various other sources. Memory management is implicit, resizing allocates at least a chunk.

◆ alloc_block

const size_t dodo::common::OctetArray::alloc_block = 32

Always allocate chunks of this size.

Any OctetArray that has been called with reserve( n > 1 ) will at least allocate 1 chunk of alloc_block bytes. Avoids frequent calls to realloc when using many append calls with small amount of data such as append( Octet ).
